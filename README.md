# 📊 Category Stats — Плагин для FunPay Cardinal

## 📋 Описание

**Category Stats** — плагин для FunPay Cardinal, который позволяет подсчитать заработок по конкретной игре, категории или ключевому слову. Сканирует все ваши продажи, группирует по периодам и выводит детальную статистику с топом категорий.

## 🌟 Возможности

| Функция | Описание |
|---------|----------|
| 🔍 **Поиск по ключевому слову** | Фильтрация продаж по названию игры, категории или товара |
| 📅 **Статистика по периодам** | Разбивка на день, неделю, месяц и всё время |
| 💰 **Подсчёт заработка** | Автоматический подсчёт суммы в разных валютах |
| ♻️ **Учёт возвратов** | Отдельная статистика по возвращённым заказам |
| 🏆 **Топ категорий** | Рейтинг самых прибыльных категорий по запросу |
| 🔄 **Обновление данных** | Мгновенное обновление статистики одной кнопкой |
| ⚡ **Прогресс сканирования** | Отображение процесса в реальном времени |
| 🔐 **Потокобезопасность** | Защита от параллельных запросов |

## 📦 Установка

1. Скачайте файл `category_stats.py` из этого репозитория
2. Откройте чат с ботом FunPay Cardinal в Telegram
3. Введите команду: `/upload_plugin`
4. Отправьте скачанный файл плагина боту
5. Перезапустите бота

> **Примечание:** Плагин не требует установки дополнительных зависимостей — используются только стандартные модули FunPay Cardinal.

## ⚙️ Использование

### Быстрый старт

1. Откройте Telegram-бота FunPay Cardinal
2. Введите команду: `/category_stats`
3. Введите название игры или ключевое слово
4. Дождитесь завершения сканирования
5. Получите детальную статистику

### Примеры запросов

Lineage 2 — все продажи по игре
Adena — конкретный товар
Dota 2 — другая игра
Gold — общий поиск по ключевому слову

## 🎛 Управление через Telegram

### Команда

/category_stats

Открывает интерфейс поиска по продажам.

### Кнопки управления

| Кнопка | Описание |
|--------|----------|
| 🔄 **Обновить** | Пересканировать продажи по текущему запросу |
| 🔍 **Новый поиск** | Начать поиск по другому ключевому слову |

## 🔧 Технические детали

- **Поиск:** Регулярные выражения с точным совпадением слов
- **Определение периодов:** Автоматический парсинг дат из HTML заказов
- **Мультивалютность:** Поддержка нескольких валют с раздельным подсчётом
- **Потокобезопасность:** Использование `threading.Lock` для защиты от параллельных запросов
- **Прогресс:** Обновление статуса каждые 5 страниц продаж
- **Ретраи:** 3 попытки при ошибках загрузки страниц
